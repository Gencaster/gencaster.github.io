A non-linear audio streaming framework for real-time radiophonic experiences and live music.

About

Gencaster is an open source platform for creating non-linear audio experiences. It rethinks the way we listen to audio: instead of playing pre-produced files, the audio is produced, mixed and delivered in real time. This allows for audio experiences with truly generated, personalized and adaptive content.

The platform creates unique real time streams per user instead of delivering the same synchronized stream to everyone (traditional radio broadcasting) or distributing static on-demand content (current streaming/podcast platforms).

Intermediate interaction between the content and the listener is enabled by the low latency of the WebRTC standard. This allows to introduce new formats generated by user preference or interaction (e.g. a weather forecast derived from GPS location). This enables non-linear and context aware compositions encompassing the full range of spoken text, synthesized sounds and musical formats.

The distribution happens via our player, which can be embedded into existing or new websites and apps.

Features

Low Latency

WebRTC allows us to stream audio with a latency of around 100ms. Traditional real time audio streams on the internet mostly run with a delay of around 20 seconds, making real time interaction or generation impossible.

User interaction

Due to the low latency we can use active or passive feedback of the listeners. These interactions can be based on anything, e.g. input masks or the current GPS location. This enables not only for dynamic but also context aware content best suited for the listener. Besides the numerical data, we can also make use of the microphone of the listener, using speech input or analyzing the environmental sounds of the listener.

Generative storytelling

Having the possibility to access properties from the user allows to create new ways of storytelling. E.g. current date, location or movement can all be used to generate new paths of a story or audio composition. We can plug any other content generation system into the audio stream.

Parallel streams

Gencaster can serve many different use cases, to either offer multiple users their personal stream or that multiple users can share one stream. It’s also imaginable to mix both or group streams. The streams are spun up and assigned to users in less than 1 second and allows for managing this assignment via e.g. an optional login.
